## About This Repository

This repository contains supplementary materials for the ECE 260 Tutorial T01.
It is intended to help students review key ideas, prepare for tutorial sessions,
and practice problem-solving strategies for continuous-time signals and systems.

The repository is organized into two main folders:

- `lecture-summaries/`: short reference summaries for selected lecture content.
- `tutorial-materials/`: tutorial handouts used or adapted for tutorial sessions.

This repository does not include the official assignment files, textbook
material, or copyrighted course documents.

## Lecture Summaries

The lecture summaries are informal study references. They are written to help
students organize ideas from lectures, connect examples to key concepts, and
review important definitions or methods.

Important note: lecture summaries are not official course materials. They should
not be treated as a replacement for the textbook, lecture slides, lecture
recordings, instructor announcements, or any material posted on the official
course website. If there is any inconsistency, the textbook, official slides,
and instructor-provided materials take priority.

## Tutorial Materials

The tutorial materials are classroom-facing handouts. A typical tutorial
handout may include:

- a brief recap of the most relevant concepts;
- selected practice questions;
- routes, hints, and solution strategies;
- intermediate derivations or MATLAB demo code when useful.

These materials are designed for learning how to approach problems, not for
copying into assignments. The tutorial questions may be selected, modified, or
adapted from related topics, and they are not guaranteed to be identical to the
official assignment questions. Always use the official assignment document when
preparing your submission.

## How to Use These Materials

A suggested workflow:

1. Watch the lecture and review the official slides or textbook section first.
2. Read the corresponding lecture summary to check the main ideas.
3. Work through the official assignment or practice problems on your own.
4. Use the tutorial materials to compare approaches, identify missing concepts,
   and improve your problem-solving route.
